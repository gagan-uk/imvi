# **imvi**

imvi is a light weight image viewer built in C++ with raylib for unix systems.

## Installation

Clone the repo:
```
git clone https://github.com/gagan-uk/imvi.git
```

After cloning the repo,open it and build the files by:
```
make install 
```

Now you can type```imvi``` in your prefred shell to open images.
